Typography & Words

We use Lato for our typeface. It can be found on Google Fonts or downloaded as font files.

General rules:

  1. Base font-size is 16px, which is defined on the body level.
  2. Consecutively, em is used for setting sizes throughout the document instead of px.
  3. If needed, increase sizes by multiples of 4px, which is 0.25em per increment.
  4. By default, the line-height is 1.5em, though it can vary depending on different circumstances.
  5. Unless customizing for certain use case, you don't really need to worry about doing the calculation as you can just use the preset.
  6. HTML represents meaning. CSS represents appearance. Basically, stick with HTML unless you are styling for purely appearance purpose.
    1. Use <em> tag if you want to express something in a different tone.
    2. Use <strong> tag if you want to emphasize or call out something.
  7. Font weights are light, regular, and bold, which equals to 300, 400, and 700 respectively.
    1. Body text is standard by default.
    2. <strong> tags are bold by default.
    3. No element is light by default. Use it for decoration purpose.

Components & States

General rules:

  1. All components don't have rounded corners.
  2. All transition duration is 0.2s unless otherwise specified for better easing.

Buttons:

  1. State changes of buttons are achieved by changing background-color to lighten($color,10%) on hover and darken($color,5%) on active.
  2. Normally, the dimension of the button is defined by padding: 0.75em 1.25em. Otherwise, just use the 0.75em vertical padding for varying width.
  3. There are color specific outline for the two buttons, overwriting the default user-agent styling and to maintain accessibility.
  4. Buttons have no borders.

Form items:

  1. State changes are achieved by changing border-color to lighten($alt-action,10%) on hover and $warning-red on warning/error.
  2. Use padding-left: 0.25em inside the form field so the cursor isn't sticking right on the left edge.

Links:

  1. State changes are similar to buttons: Change color to lighten($color,10%) on hover and darken($color,5%) on active.
  2. Keep the text-decoration: underline, which comes with Bootstrap 3.0 by default.
  3. For links in text, maintain the regular font-weight, but use font-weight: 700 for navigation links.
  4. Normally, use the two main colors for links, but you can use other colors if the situation requires.
